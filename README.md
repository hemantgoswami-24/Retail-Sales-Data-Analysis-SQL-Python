# SQL-Pandas-Retail-Analysis
End-to-end Retail Shop Sales Analysis using **MySQL + Python (Pandas & Matplotlib)**.  
This project includes:

- Total Sales, Profit, and Quantity Analysis
- Category-wise, Product-wise, and City-wise breakdown
- Monthly sales trends
- Visualizations: Bar charts & Pie charts
- SQL integration with Pandas for automated data extraction

**Note:** Sample data is included in `RetailShopDB.sql`. Python script calculates all metrics dynamically.

---

## Usage

1. Run `RetailShopDB.sql` to create database and tables.  
2. Update Python script with your MySQL credentials.  
3. Run `Retail_Sales_Analysis.py` to generate analysis & visualizations.
